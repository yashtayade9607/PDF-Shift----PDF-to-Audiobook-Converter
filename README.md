📄 PDF Shift
PDF Shift is a versatile web-based tool built using HTML, CSS, JavaScript, and PHP that provides seamless conversions between different document and media formats. It is designed to enhance accessibility, productivity, and user convenience by offering a wide range of file transformation features — all in one place.

🚀 Features
🔊 PDF to Audiobook – Convert PDFs into audio files for easy listening.

📝 Audiobook to PDF – Generate readable text or PDFs from audio content.

🗣️ Text to Audiobook – Turn plain text into speech using TTS (Text-to-Speech).

📘 Audiobook to Text – Extract and transcribe spoken words from audio files.

🖼️ PDF to JPEG – Convert PDF pages into high-quality image files.

📎 PDF Merger – Combine multiple PDF files into a single, unified document.

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript

Backend: PHP

📌 Use Cases
Accessibility support for visually impaired users

Creating study materials or voice notes

Extracting information from audiobooks or spoken lectures

Managing and merging PDF files efficiently

📥 Getting Started
Clone the repo and start a local PHP server:

1) git clone https://github.com/yashtayade9607/PDF-Shift----PDF-to-Audiobook-Converter.git
2) cd PDF Shift
3) Open XAMPP
4) Start Apache Server and MySQL
5) Open http://localhost:8000/foldername/index.php in your browser.
